## How to Run
```
host> $ docker build -t automanation .
host> $ docker run --rm -it automanation
```

## How to Play
```bash
docker> $ ruby main.rb

press <h> to move left
press <j> to move down
press <k> to move up
press <l> to move right
press <M> to show Menu (no items...)
press <L> to show List (of objects on current field)
press <R> to resporn when dead (hit the wall to take damage)
```
